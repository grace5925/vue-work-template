# vue-work-template

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
yarn run serve
```

### Compiles and minifies for production

```bash
yarn run build
```

### Lints and fixes files

```bash
yarn run lint
```

## 目录结构

    ├── README.md                       项目介绍
    ├── src
    │     ├── api
    │     ├── assets                          资源目录，这里的资源会被wabpack构建
    │     │     ├── images
    │     └── main.js                         入口文件
    ├── public                          纯静态资源，不会被wabpack构建。
    └──vue.config.js                    配置文件
